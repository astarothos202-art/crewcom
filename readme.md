=============================================
ИНСТРУКЦИЯ ПО НАСТРОЙКЕ GOOGLE FORMS
=============================================

ДЛЯ РАБОТЫ ФОРМЫ НЕОБХОДИМО ВЫПОЛНИТЬ СЛЕДУЮЩИЕ ШАГИ:

ШАГ 1: СОЗДАЙТЕ GOOGLE ФОРМУ
------------------------------------------------
1. Перейдите на https://forms.google.com
2. Создайте новую форму
3. Добавьте следующие поля (обязательно в этом порядке):
   - Поле с вопросом "Ваше имя" (тип: текст)
   - Поле с вопросом "Email или Telegram" (тип: текст)
   - Поле с вопросом "Опишите ваш проект..." (тип: текст (абзац))
4. Сохраните форму

ШАГ 2: ПОЛУЧИТЕ ID ФОРМЫ
------------------------------------------------
1. Откройте созданную форму
2. Посмотрите на URL в браузере, он выглядит так:
   https://docs.google.com/forms/d/e/1FAIpQLSf9GxK8J7z5m3N2p4R6v8t9w0x1y2z3a4b5c6d7e8f9g0h1i2j3k4l5m6n7o8p9/viewform
3. Скопируйте ID формы - это длинная строка между /d/e/ и /viewform:
   1FAIpQLSf9GxK8J7z5m3N2p4R6v8t9w0x1y2z3a4b5c6d7e8f9g0h1i2j3k4l5m6n7o8p9

ШАГ 3: ПОЛУЧИТЕ ID ПОЛЕЙ (entry IDs)
------------------------------------------------
1. Откройте форму в режиме предпросмотра (нажмите на "глаз" в правом верхнем углу)
2. Нажмите F12 (или Ctrl+Shift+I) чтобы открыть инструменты разработчика
3. Найдите вкладку "Elements" (Элементы)
4. Нажмите Ctrl+F и введите "entry."
5. Вы увидите атрибуты name у полей, например:
   - Для первого поля: name="entry.123456789"
   - Для второго поля: name="entry.987654321"
   - Для третьего поля: name="entry.456789123"
6. Скопируйте эти ID для каждого поля

ШАГ 4: ЗАМЕНИТЕ ЗНАЧЕНИЯ В ФАЙЛЕ index.html
------------------------------------------------
Откройте файл index.html и найдите строки (примерно строка 565-570):

const FORM_ID = 'e/1FAIpQLSf9GxK8J7z5m3N2p4R6v8t9w0x1y2z3a4b5c6d7e8f9g0h1i2j3k4l5m6n7o8p9';

const FIELD_IDS = {
    name: 'entry.123456789',
    email: 'entry.987654321',
    message: 'entry.456789123'
};

Замените эти значения на свои:
- FORM_ID: вставьте ID вашей формы (БЕЗ /e/ в начале, только сама строка)
- name: вставьте ID поля для имени
- email: вставьте ID поля для email
- message: вставьте ID поля для сообщения

ШАГ 5: ПРОВЕРЬТЕ РАБОТУ
------------------------------------------------
1. Откройте index.html в браузере
2. Заполните форму и отправьте
3. Должно появиться сообщение об успешной отправке
4. Проверьте Google Форму - ответы должны появиться во вкладке "Ответы"

ПРОБЛЕМЫ И РЕШЕНИЯ:
------------------------------------------------
- Если форма не отправляется: проверьте правильность ID
- Если CORS-ошибка: это нормально для метода no-cors
- Если ответы не приходят: попробуйте создать новую форму и повторить шаги

ВАЖНО: Форма должна быть доступна для всех (в настройках формы выберите 
"Принимать ответы" и убедитесь, что форма не требует авторизации).
